# Discord youtube music bot

<center>Discord bot for search and play songs from youtube.</center>

## Features

- [ ] Help command
- [ ] Search songs
- [ ] Play songs
- [ ] Repeat
- [ ] Queue

## Configuration

Create and fill `.env` file in root folder.

Options:
- `TOKEN` - bot token;
- `CLIENT_ID` - application id;
- `GUILD_ID` - server id;
- `LOG_TIME_FORMAT` - log datetime format (date-fns).
